# 临时的ip配置

### 1、指定ip地址

```shell
ifconfig eth0 192.168.1.10/24  # 指定ip地址

ifconfig etho # 查询ip

```

### 2、指定网关（联网的）
```shell
route add default gw 192.168.1.10 # 指定网关

netstat -nr # 查询是否生效
```

### 3、添加静态路由

> 命令添加

```shell
# /24是添加网络地址，/32是添加主机地址 gw是添加网段地址
route add -net 192.168.1.0/24 gw 192.168.1.100 eth0

route -n # 查询路由
```
