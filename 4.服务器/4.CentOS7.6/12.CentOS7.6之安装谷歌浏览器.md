# 安装配置

>1、安装运行环境

```shell
yum -y install liberation-fonts libappindicator-gtk3
```

>2、安装

```shell
cd /usr/local

rpm -ivh google-chrome-stable*
```

>3、修改配置
```
vim /etc/selinux/config
```

```
SELINUX=disabled
```

```shell
vim /usr/share/applications/google-chrome.desktop
```

```shell
Exec=/usr/bin/google-chrome-stable %U --no-sandbox
```
