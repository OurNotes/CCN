'esp8266引脚说明：'

| pin | 名称 | 注释 |
| :-: | :-: | :-: |
| 1 | 3V3| 供电3.3V |
| 2 | RX | UART |
| 3 | IO16 | 这个脚还有一个名称叫RESET |
| 4 | IO0 | GPIO0 |
| 4 | EN | 又名CHIP_EN/CH_PD |
| 4 | IO2 | GPIO2 |
| 4 | TX | UART |
| 4 | GND | 接地，电源负极 |
