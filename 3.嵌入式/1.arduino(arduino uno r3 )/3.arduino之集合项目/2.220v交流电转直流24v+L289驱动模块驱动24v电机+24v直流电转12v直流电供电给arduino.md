# <a name="arduino-03" href="#" >写入程序</a>

```c
int IN1 = 1;
int IN2 = 2;   
int PWM = 3;  
 
unsigned long time = 1200;
int value = 128;
 
void setup(){
    pinMode(IN1, OUTPUT);
    pinMode(IN2, OUTPUT);
 
    digitalWrite(IN1, HIGH);
    digitalWrite(IN2, LOW);
}
 
void loop(){
    analogWrite(PWM, value);
    delay(time);
}
```
