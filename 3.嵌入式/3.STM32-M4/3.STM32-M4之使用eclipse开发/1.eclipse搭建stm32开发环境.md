```

变量值：C:\Software\openocd-0.10.0

```

> 4、配置变量环境：gcc-arm

```
变量名：ARM_HOME
变量值：C:\Software\gcc-arm-none-eabi-8-2018-q4-major-win32

path后加：%ARM_HOME%\bin
```
