# <a name="java-02" href="#" >配置</a>

>配置server.xml
```xml
<Connector port="8089"
           protocol="org.apache.coyote.http11.Http11AprProtocol"
           connectionTimeout="20000"
           maxConnections="2000"
           maxThreads="500"
           acceptCount="500"
           minSpareThreads="100"
           compression="true"
           compressionMinSize="2048"
           redirectPort="8443" />
```
