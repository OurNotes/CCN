# <a name="go-01" href="#" >下载安装</a>

> 运行cmd，输入命令

```shell
set GO111MODULE=on
set GOPROXY=https://goproxy.io

go get -u github.com/spf13/cobra/cobra
go install github.com/spf13/cobra/cobra

```
