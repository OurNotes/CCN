# <a name="vultr-02" href="#" >配置</a>

> 开放防火墙

```shell

firewall-cmd --permanent --zone=public --add-port=42301/tcp
firewall-cmd --reload
firewall-cmd --zone=public --query-port=42301/tcp
```
