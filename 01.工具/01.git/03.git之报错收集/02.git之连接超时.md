# 报错

```
Failed to connect to github.com port
443: Timed out
```

# 解决

> git的安装目录下的/etc/ssh/ssh_config文件添加
```
Host github.com
User git
Hostname ssh.github.com
PreferredAuthentications publickey
IdentityFile ~/.ssh/id_rsa
Port 443
```
