> 写代码

- test.go

```go
package main

import (
	"github.com/gin-gonic/gin"
	"net/http"
)

func main() {
	r := gin.Default() 
	r.Static("/html","./view1")
	r.StaticFS("/html2",http.Dir("./view2"))
	r.Run() // 启动并监听8080端口
}

```

# <a name="go-02" href="#" >编译</a>

> 按快捷键：<kbd>Ctrl</kbd>+<kbd>`</kbd>,vs code进入终端输入命令。

```shell
set GO111MODULE=on
set GOPROXY=https://goproxy.io

go mod init test
go mod tidy

```


> 运行

```shell

cd start

go run test.go
```
