# 配置

> 配置环境变量

```
变量名：YARM_HOME
变量值：C:\Software\Yarn
```

Path的值后面加：;%YARM_HOME%;%YARM_HOME%\bin\
# 测试

- cmd输入命令

```shell
yarn -version
```
