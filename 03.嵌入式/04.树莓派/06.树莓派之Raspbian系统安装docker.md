总操作流程：
- 1、下载安装
- 2、配置
- 3、测试

***

# 下载安装

```shell
su

apt-get update

curl -sSL https://test.docker.com  | sh

```


# 配置


```shell

# 设置为自动启动
sudo systemctl enable docker

# 启动守护程序
sudo systemctl start docker
```


# 测试

```
# 启用客户端
sudo usermod -aG docker pi

docker --version
```
