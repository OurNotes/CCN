# <a name="STM-M4-03" href="#" >修改文件</a>

### <a name="STM-M4-03-01" href="#" >1、修改main.c文件</a>

<details>
<summary>代码</summary>

```c
#include "sys.h"
#include "usart.h"
#include "delay.h"
int main(void)
{
    u8 t=0;
    Stm32_Clock_Init(336,8,2,7);//初始化时钟为 168Mhz
    delay_init(168); //初始化延时函数
    uart_init(84,115200); //串口初始化为 115200
    while(1)
    {
        printf("t:%d\r\n",t);
        delay_ms(500);
        t++;
    }
}

```
