# <a name="arduino-02" href="#" >写入程序</a>
```c
int led = 13; // 定义针脚号，数字类型为整型
/**
 * 对Arduino电路板或相关状态进行初始化方法
 */
void setup() {
  pinMode(led, OUTPUT); // 设定13号针脚为输出状态，
}
/**
 *  系统调用，无限循环方法
 */
void loop() {
  digitalWrite(led, HIGH);// 向13号针脚输出值为高电压状态，
}
```
