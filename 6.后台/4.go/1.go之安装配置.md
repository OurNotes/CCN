- 2、配置
- 3、测试

# 下载安装

[![](https://img.shields.io/badge/官网-go-red.svg "官网 go")](https://golang.google.cn/dl/)

- 注意：应该下载压缩包，因为安装程序会指定工作目录

# 配置

- 新建

```c
变量名：GOROOT
变量值: C:\Software\Go

变量名：GOBIN
变量值：C:\Software\Go\bin


变量名：GOPATH
变量值：D:\goWork
```

- 添加到path

```c
;%GOROOT%;%GOPATH%;%GOBIN%
```

# 测试

- cmd下输入命令

```c
go env
```
