总操作流程：
- 1、创建create文件
- 2、测试

> 本教程基于：cobra之初始化

***

# 创建文件

```shell
# vs code打开项目，在控制台输入命令
cobra add create
```

# 测试

```shell
# vs code打开项目，在控制台输入命令
set GO111MODULE=on
set GOPROXY=https://goproxy.io

go mod init demo
go mod tidy

go run main.go create
```

- 成功标志

```
C:\Users\admin\Desktop\demo>go run main.go create
create called
```
