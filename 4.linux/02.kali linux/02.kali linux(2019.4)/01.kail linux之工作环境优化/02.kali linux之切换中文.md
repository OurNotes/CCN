总操作流程：
- 1、选择语言
- 2、安装文字
- 3、看效果

***

# 选择语言

```shell
dpkg-reconfigure locales
```

- 进入图形界面，选中en_US.UTF-8 UTF-8和zh_CN.UTF-8 UTF-8（空格是选择，tab是切换，*是选中）并将zh_US.UTF-8选为默认

![](image/2-1.png)

# 安装文字

```
apt-get install xfonts-intl-chinese
apt-get install ttf-wqy-microhei

# win10 模式
kali-undercover

reboot
```
