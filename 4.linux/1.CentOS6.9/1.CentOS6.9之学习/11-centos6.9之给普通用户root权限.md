# <a name="Linux-02" href="#" >编辑修改配置文件</a>
```shell
visudo 或者 vi /etc/sudoers
```
修改内容：
在“root  ALL=(ALL)   ALL”这一行下面，再加入：
```shell
#Give dk a root authority
dk    ALL=(ALL)       ALL
```
# <a name="Linux-03" href="#" >切换到普通用户</a>
```shell
su dk
```
