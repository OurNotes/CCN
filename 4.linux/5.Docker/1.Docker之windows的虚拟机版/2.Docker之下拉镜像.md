# <a name="docker-03" href="#" >测试</a>

> 1、切换到centos:7.6.1810容器里

```
docker run -i -t centos:7.6.1810 /bin/bash
```

> 2、退出容器回到docker

```
exit
```

- <kbd>Ctrl</kbd>+<kbd>D</kbd>快捷键方式
